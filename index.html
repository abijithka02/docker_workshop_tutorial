What is Docker?

Docker is an open-source platform for developing, shipping, and running applications in lightweight, portable containers.

Why Use Docker?

  • Ensures consistency across different environments.
  • Efficient resource utilization.
  • Rapid application deployment.
  • Easy dependency management.

What is Docker Hub?

Docker Hub is a cloud-based registry that allows you to find and share container images.

Steps to Pull and Execute a Python Program using Docker

1. Verify Docker Installation

$ docker --version

2. Pull the Python Image

$ docker pull python

3. Run a Python Container

$ docker run -it --name python-container python

4. Verify Python Installation inside Container

# python --version

5. Create a Python Script (inside the container)

# echo 'print("Hello, Docker!")' > script.py

6. Run the Python Script

# python script.py

7. Exit the Container

# exit
